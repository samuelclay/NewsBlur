Web Feeds

Follow any website, even ones that don't publish an RSS feed.

Web feed creation showing page conversion

How it works

  1. Paste any URL into the "Add Site" dialog
  2. If no RSS feed is found, NewsBlur offers to create a Web Feed
  3. NewsBlur monitors the page for changes automatically
  4. New content appears in your feed just like regular RSS
  5. Works with social media profiles, job boards, product pages, and more

What sites can I follow with Web Feeds?

Any public web page. Twitter profiles, Reddit threads, GitHub repos, product pages, job boards, government sites, and more.

How often are Web Feeds checked?

The same as regular feeds. Premium and Pro users get faster update frequencies.

Is this different from RSS?

Yes. Web Feeds monitor a page's HTML for changes, while RSS reads a structured feed file. Web Feeds work even when sites don't offer RSS.
