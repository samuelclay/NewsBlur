Full-Text Search

Search across all your subscriptions with Elasticsearch-powered full-text search.

Search results across subscriptions

How it works

  1. Press the search icon or use the keyboard shortcut
  2. Type keywords, phrases, or combinations
  3. Search within a single feed, a folder, or across everything
  4. Results show matching stories with highlighted terms
  5. Save searches as feeds for ongoing monitoring

What plan do I need for search?

Full-text search requires Premium ($36/year) or higher.

Can I search saved stories?

Yes, saved stories and their tags are fully searchable.

How far back does search go?

On Premium, search covers your recent stories. On Premium Archive, every story ever archived is searchable.

Can I save a search as a feed?

Yes, save any search as a feed to get ongoing results.
